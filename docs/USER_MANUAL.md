## 1. ภาพรวมระบบ

### 1.1 วัตถุประสงค์
ระบบ Logistics Delivery KPI Management ใช้สำหรับ:
- ติดตามและวิเคราะห์ข้อมูลการจัดส่งสินค้า
- คำนวณ KPI ด้านโลจิสติกส์
- วางแผนทรัพยากร (รถ, คน) ล่วงหน้า
- พยากรณ์ความต้องการทรัพยากรในอนาคต

### 1.2 เมนูหลัก
| เมนู | หน้าที่ |
|------|--------|
| 📊 Dashboard | แสดงภาพรวม KPI และกราฟวิเคราะห์ |
| 📦 Delivery Track | ติดตามสถานะการจัดส่ง |
| 📈 Forecast | พยากรณ์ความต้องการทรัพยากร |
| 🏢 ทรัพยากรสาขา | ตั้งค่าทรัพยากรแต่ละสาขา |
| 📥 นำเข้าข้อมูล | Import ข้อมูล Delivery |

## 3. การนำเข้าข้อมูล Delivery

### 3.1 เตรียมไฟล์ Excel
**รูปแบบไฟล์ที่รองรับ:** `.xlsx`, `.xls`

**คอลัมน์ที่จำเป็น:**
| คอลัมน์ | คำอธิบาย | ตัวอย่าง |
|---------|----------|----------|
| วันที่ | วันที่จัดส่ง | 15/01/2024 |
| สาขา | ชื่อสาขา | สาขาพระราม 2 |
| เลข Invoice | หมายเลข Invoice | INV-2024-001 |
| จำนวน (กล่อง) | ปริมาณสินค้า | 150 |
| สถานะ | สถานะการจัดส่ง | สำเร็จ |

### 3.2 ขั้นตอนการนำเข้า

1. **เข้าเมนู "📥 นำเข้าข้อมูล"**
   
2. **เลือกไฟล์ Excel**
   - คลิกปุ่ม "เลือกไฟล์" หรือลากไฟล์มาวาง
   - ระบบจะแสดงตัวอย่างข้อมูล

3. **ตรวจสอบการ Mapping คอลัมน์**
   - ตรวจสอบว่าคอลัมน์ตรงกับข้อมูลที่ต้องการ
   - แก้ไข Mapping หากไม่ถูกต้อง

4. **คลิก "นำเข้าข้อมูล"**
   - รอระบบประมวลผล
   - ตรวจสอบผลลัพธ์ (จำนวนรายการที่นำเข้าสำเร็จ/ล้มเหลว)

### 3.3 ข้อควรระวัง
- ⚠️ ตรวจสอบรูปแบบวันที่ให้ถูกต้อง
- ⚠️ ชื่อสาขาต้องตรงกับที่มีในระบบ
- ⚠️ ไม่ควรมีแถวว่างในไฟล์

## 4. การตั้งค่าทรัพยากรสาขา

### 4.1 วัตถุประสงค์
บันทึกข้อมูลทรัพยากรปัจจุบันของแต่ละสาขา เพื่อใช้ในการคำนวณ Forecast

### 4.2 ขั้นตอนการตั้งค่า

1. **เข้าเมนู "🏢 ทรัพยากรสาขา"**

2. **เลือกสาขา**
   - เลือกจาก Dropdown "เลือกสาขา"

3. **เลือกเดือนอ้างอิง**
   - เลือกเดือนที่มีข้อมูล Delivery สำหรับคำนวณค่าเฉลี่ย

4. **กรอกข้อมูลทรัพยากร**

   **🚚 รถขนส่ง:**
   | ฟิลด์ | คำอธิบาย | ตัวอย่าง |
   |-------|----------|----------|
   | จำนวนรถประจำสาขา | รถที่มีประจำ | 14 คัน |
   | รอบ/วัน | จำนวนเที่ยวต่อวัน | 2 รอบ |
   | ค่าเช่ารถเพิ่ม/วัน | ค่าเช่ารถถ้าต้องเช่าเพิ่ม | 2,500 บาท |

   **👷 กำลังคน:**
   | ฟิลด์ | คำอธิบาย | ตัวอย่าง |
   |-------|----------|----------|
   | Loaders | พนักงานขนถ่าย | 10 คน |
   | Checkers | พนักงานตรวจนับ | 10 คน |
   | Admin | พนักงานธุรการ | 4 คน |
   | ชม.ทำงาน/วัน | ชั่วโมงทำงานปกติ | 8 ชม. |

   **💰 ค่าแรง:**
   | ฟิลด์ | คำอธิบาย | ตัวอย่าง |
   |-------|----------|----------|
   | ค่าแรง Loader/วัน | | 450 บาท |
   | ค่าแรง Checker/วัน | | 400 บาท |
   | ค่าแรง Admin/วัน | | 500 บาท |

5. **ตรวจสอบค่าที่คำนวณอัตโนมัติ**
   
   ระบบจะคำนวณให้:
   - **ความจุเฉลี่ย (กล่อง/เที่ยว):** = ยอดรวมเดือน ÷ (จำนวนรถ × รอบ/วัน)
   - **ความเร็วขนถ่ายเฉลี่ย (กล่อง/ชม./คน):** = ยอดรวมเดือน ÷ (Loaders × ชม.ทำงาน × วันทำงาน)

6. **คลิก "💾 บันทึก"**
   - ระบบจะบันทึกข้อมูลและค่าที่คำนวณได้

### 4.3 ตัวอย่างหน้าจอ

```
┌─────────────────────────────────────────────────────────────────┐
│  🏢 ทรัพยากรสาขา                                                 │
├─────────────────────────────────────────────────────────────────┤
│  สาขา: [สาขาพระราม 2 ▼]     เดือน: [มกราคม 2024 ▼]              │
├─────────────────────────────────────────────────────────────────┤
│  🚚 รถขนส่ง                                                      │
│  ├─ จำนวนรถ: [14] คัน                                            │
│  ├─ รอบ/วัน: [2] รอบ                                             │
│  └─ ค่าเช่าเพิ่ม: [2,500] บาท/วัน                                 │
│                                                                 │
│  📊 ความจุเฉลี่ย (คำนวณอัตโนมัติ): 4,907 ÷ (14 × 2) = 175 กล่อง/เที่ยว │
├─────────────────────────────────────────────────────────────────┤
│  👷 กำลังคน                                                      │
│  ├─ Loaders: [10] คน   Checkers: [10] คน   Admin: [4] คน        │
│  └─ ชม.ทำงาน/วัน: [8] ชม.                                        │
│                                                                 │
│  📊 ความเร็วขนถ่ายเฉลี่ย: 4,907 ÷ (10 × 8) = 61 กล่อง/ชม./คน      │
├─────────────────────────────────────────────────────────────────┤
│                                          [💾 บันทึก]            │
└─────────────────────────────────────────────────────────────────┘
```

## 5. การใช้งาน Forecast

### 5.1 วัตถุประสงค์
พยากรณ์ความต้องการทรัพยากร (รถ, คน, ค่าใช้จ่าย) สำหรับเดือนถัดไป

### 5.2 ข้อกำหนดก่อนใช้งาน
✅ ต้องนำเข้าข้อมูล Delivery แล้ว  
✅ ต้องตั้งค่าทรัพยากรสาขาแล้ว

### 5.3 ขั้นตอนการใช้งาน

1. **เข้าเมนู "📈 Forecast"**

2. **เลือกสาขา**
   - ระบบจะโหลดข้อมูลทรัพยากรของสาขานั้นมาแสดง

3. **ตั้งค่า Forecast**

   **📅 เปรียบเทียบยอดหลายเดือน:**
   - เลือกเดือนที่ต้องการเปรียบเทียบ
   - เลือกเดือนที่จะใช้เป็นฐาน Forecast
   - กำหนด % การเติบโต (ถ้ามี)
   - กำหนดวันทำงานของเดือน Forecast

   **🚚 รถขนส่ง:**
   - ค่าความจุจะแสดงจาก "ทรัพยากรสาขา" (สีเขียว ✓)
   - ปรับจำนวนรอบ/วัน ได้ตามต้องการ

   **👷 กำลังคน:**
   - ค่าความเร็วขนถ่ายจะแสดงจาก "ทรัพยากรสาขา" (สีเขียว ✓)
   - ปรับอัตราส่วน Checker:Loader ได้

   **⏰ OT และ Part-time:**
   - กำหนดค่า OT ต่อชั่วโมง
   - กำหนดค่าแรง Part-time ต่อวัน

4. **ดูผลลัพธ์ Forecast**

   ระบบจะแสดง:
   - **ยอด Forecast:** ปริมาณสินค้าที่คาดการณ์
   - **รถที่ต้องใช้:** จำนวนรถและค่าเช่าเพิ่ม (ถ้ามี)
   - **คนที่ต้องใช้:** จำนวน Loaders/Checkers และค่าใช้จ่าย
   - **คำแนะนำ OT/Part-time:** เปรียบเทียบต้นทุน

### 5.4 ตัวอย่างผลลัพธ์

```
┌─────────────────────────────────────────────────────────────────┐
│  📈 ผลการพยากรณ์ - สาขาพระราม 2                                   │
├─────────────────────────────────────────────────────────────────┤
│  📦 ยอด Forecast                                                 │
│  ├─ ยอดฐาน: 30,890 กล่อง (ม.ค. 2024)                             │
│  ├─ + การเติบโต: 10%                                             │
│  └─ ยอด Forecast: 33,979 กล่อง (1,359 กล่อง/วัน)                  │
├─────────────────────────────────────────────────────────────────┤
│  🚚 รถขนส่ง                                                      │
│  ├─ ต้องใช้: 18 คัน (8 เที่ยว/วัน @ 175 กล่อง/เที่ยว)             │
│  ├─ มีประจำ: 14 คัน                                              │
│  └─ ❗ ต้องเช่าเพิ่ม: 4 คัน (10,000 บาท/วัน)                       │
├─────────────────────────────────────────────────────────────────┤
│  👷 กำลังคน                                                      │
│  ├─ ต้องใช้ Loaders: 22 คน (@ 61 กล่อง/ชม./คน)                    │
│  ├─ มีประจำ: 10 คน                                               │
│  └─ ❗ ต้องเพิ่ม: 12 คน                                            │
├─────────────────────────────────────────────────────────────────┤
│  💡 คำแนะนำ                                                      │
│  ├─ OT Cost: 8,400 บาท/วัน                                       │
│  ├─ Part-time Cost: 5,400 บาท/วัน                                │
│  └─ ✅ แนะนำ: Part-time (ประหยัดกว่า 3,000 บาท/วัน)               │
└─────────────────────────────────────────────────────────────────┘
```

## 6. การดูรายงาน Dashboard

### 6.1 ภาพรวม KPI
- **ยอดจัดส่งรวม:** ปริมาณสินค้าทั้งหมด
- **จำนวน Invoice:** จำนวนใบส่งของ
- **อัตราความสำเร็จ:** % การจัดส่งสำเร็จ
- **เปรียบเทียบเดือนก่อน:** % การเปลี่ยนแปลง

### 6.2 กราฟวิเคราะห์
- กราฟแนวโน้มรายวัน/รายสัปดาห์
- กราฟเปรียบเทียบสาขา
- กราฟประสิทธิภาพการจัดส่ง

## 7. การจัดการข้อมูลอื่นๆ
